AluraBooks

This project was undertaken at Alura programming school. It constitutes an exercise aimed at creating a responsive webpage. While originally intended to be developed solely with CSS and HTML, I opted to incorporate React and SCSS into the project as well. Knowledge of React, HTML, and SCSS was utilized in the construction of this project.

Project Link

Project Overview

Our project aims to develop a responsive book page, catering to users accessing it from various devices such as smartphones, tablets, and PCs. The primary focus is on ensuring a seamless user experience across different screen sizes and resolutions.
